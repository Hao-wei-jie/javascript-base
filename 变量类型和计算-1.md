**知识点：**

1. 变量类型
   - 值类型 VS 引用类型
      - 值类型：每个变量都会存储自己的值，不会相互影响。例：

      ```
        var a = 100
        var b = a
        a = 200
        console.log(b) // 100
      ```

      - 引用类型：

      ```
        var a = {age: 20}
        var b = a
        b.age = 21
        console.log(a.age) // 21
      ```
